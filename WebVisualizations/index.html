Welcome to a weather analysis!

The graphs on these webpages demonstrate how different weather effects correlate to changes in latitude.

This project was a combination of a number of our processes. We first utilized some random generation to find various cities all across the world. From there, we then utilized the weather api to get various details of the weather of each city. With these weather details in hand, we then used matplotlib to create graphs to demonstrate whether there were correlations between the latitude of the city and that particular weather pattern. The links above will take you to the four graphs: temperature, humidity, cloudiness, and wind speed. Additionally, there is a place where you can see all four of the graphs. Finally, you can look at the actual data which was analyzed. Thanks for checking this out!
